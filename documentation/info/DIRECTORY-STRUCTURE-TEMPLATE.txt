================================================================================
FILE FORMAT SPECIFICATIONS
================================================================================

IMAGES:
  CAD Renders:
    - Format: PNG or JPEG (PNG preferred for transparency)
    - Dimensions: 1920x1080 pixels minimum
    - Files: 8 total (front, side, top, isometric, 3x closeups, exploded)
    - Location: assets/cad-renders/

  Photos (Build Progress):
    - Format: WebP (80% smaller than JPEG)
    - Dimensions: Max 2000x1500px
    - File size: <500KB per image
    - Files: 30+ from build images folder
    - Location: assets/photos/build-progress/
    - Keep JPEG backups for reference

  Component Photos:
    - Format: WebP
    - Dimensions: Max 2000x1500px
    - File size: <500KB
    - Files: 5-8 closeups of key components
    - Location: assets/photos/components/

VIDEOS:
  Format: H.264 codec, MP4 container
  Resolution: 1920x1080 (1080p)
  Bitrate: 3-5 Mbps
  Framerate: 30fps
  File size: <100MB per video
  Files: 6-8 assembly walkthroughs
  Location: assets/videos/
  Also create .jpg thumbnails (1280x720)

DATA FILES (JSON):
  Format: Valid JSON
  Charset: UTF-8
  Location: public/data/
  Files:
    - bom.json (500+ parts)
    - timeline.json (20+ events)
    - photos.json (30+ photos)
    - videos.json (6+ videos)
    - components.json (10+ components)

MARKDOWN FILES:
  Format: GitHub-flavored Markdown
  Charset: UTF-8
  Front matter: YAML (title, description, date, image)
  Location: content/
  Files: 1 home + 6 phases + 5 technical + 1 about = 13 total
