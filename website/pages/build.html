PHASE 1

Illumination Path

The illumination path generates and shapes the light sheet that excites fluorophores in the sample. This phase covers the laser source, beam expansion optics, galvanometer mirrors for remote scanning, and the optics that form the final light sheet.

Key Principles

  • • Remote Scanning: The galvo mirrors deflect the beam before it enters the objective, allowing the light sheet to scan without moving the sample.
  • • Beam Expansion: The laser beam is expanded to fill the back aperture of the illumination objective for optimal light sheet thickness.
  • • Cylindrical Optics: Convert the Gaussian beam into a light sheet profile.

Assembly Steps

1

Mount the Laser Source

Secure the 488nm laser to the optical table using appropriate posts and clamps. Ensure the beam height matches your optical axis (typically 4" or 100mm).

Safety: Always use appropriate laser safety eyewear when aligning.
2

Install Beam Expansion Optics

Set up the beam expander to achieve the target beam diameter. Use a beam profiler to verify Gaussian profile and collimation.

Tip: Target beam diameter should fill ~80% of the objective back aperture.
3

Mount Galvanometer Assembly

Position the galvo mirrors at the conjugate plane. The mirror pivot should be at the pupil plane of the scan lens system.

4

Install Scan and Tube Lenses

Set up the 4f relay system with scan lens and tube lens to create telecentric scanning. Verify the focal lengths are matched for the desired magnification.

5

Add Cylindrical Lens for Light Sheet

Position the cylindrical lens to create the light sheet profile. The light sheet thickness is determined by the NA at the sample plane.

PHASE 2

Imaging Path

The imaging path collects fluorescence from the sample and forms an image on the camera sensor. This includes the detection objective, tube lens, emission filters, and camera mounting.

Key Considerations

  • • Remote Focusing: A remote mirror allows axial scanning without moving the sample or objective.
  • • High NA Collection: Use a high-NA detection objective to maximize fluorescence collection efficiency.
  • • Filter Selection: Emission filters must match the voltage indicator emission spectrum while blocking excitation light.

PHASE 3

Electronics Integration

The electronics phase connects all components for synchronized control. This includes the DAQ system, piezo controllers, galvo drivers, and signal routing.

System Architecture

The electronics system uses a National Instruments DAQ for generating analog control signals and digital triggers. Precise timing synchronization is critical for remote scanning.

PHASE 4

Software Setup

The software phase covers control software configuration, data acquisition pipelines, and real-time processing setup for voltage imaging experiments.

Software Stack

  • • LabVIEW/MATLAB: DAQ control and waveform generation
  • • HCImage: Camera acquisition software
  • • Python: Data processing and analysis pipelines
